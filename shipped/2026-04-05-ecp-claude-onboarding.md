## What It Is

A structured onboarding system for getting the ECP marketing team up and running with Claude AI. Includes CLAUDE.md project instructions, setup guides, and a prompt library.

## Use Case

Reduces ramp time for new Claude users on the team. Ensures consistent usage patterns, shared context files, and a common prompt library across the team. Designed to evolve as Claude usage at ECP matures.
